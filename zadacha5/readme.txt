На странице есть видео, которое загружается из youtube.com, есть картинки(находятся в папке imgs) - при клике на них мышкой увеличиваются в размере и есть кнопки авторизации(при нажатии происходит переход на страницу регистрации signup.php) и входа(переход на страницу авторизации login.php). Есть ссылка для перехода на страницу с таблицей tab.php содержащей данные авторизованных клиентов. Таблица базы данных в формате zad5_db.sql экспортирована в папку zadacha5.
